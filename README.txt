Description
===========

An installable Ploomcake theme based on redomino.css3theme, a mobile responsive theme for Plone (https://github.com/redomino/redomino.css3theme).

NOTE: this is not a standalone package, thus you cannot install it individually. 
Please visit http://www.ploomcake.org/en/download to get Ploomcake.

Ploomcake is a Plone distribution that allows you to create a ready-to-use Plone website, collaboration portal 
or intranet, already connfigured with many addons.
